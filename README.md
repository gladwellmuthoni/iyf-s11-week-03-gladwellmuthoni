## Project Description
I developed a multi-page, responsive personal portfolio website from scratch using HTML5 and CSS3. The project serves as a digital hub to showcase my technical skills, professional background, and completed projects. I built this to master foundational web architecture, CSS layout systems, and professional version control workflows.

## Technologies Used
- HTML5
- CSS3
- GitHub
- Git

## Features
-Multi-page navigation: Smooth movement between Home, About, Projects, and Contact sections.

-Responsive Project Gallery: A grid-based layout for showcasing technical work with descriptive content.

-Functional Contact Form: A dedicated page for user inquiries.

-Consistent Design: Centralized styling via external CSS with page-specific overrides.

-Using features like code space.
